## Installation

Get Lord Commander installed and ready for use in your development workflow.

### Step 1: Install Lord Commander

<AccordionGroup>
  <Accordion icon="download" title="Install via npm">
    Install Lord Commander globally to use it across all your projects:
    
    ```bash
    npm install -g @caedonai/lord-commander
    ```
    
    Or install locally in your project:
    
    ```bash
    npm install --save-dev @caedonai/lord-commander
    ```
  </Accordion>
  <Accordion icon="box" title="Install via pnpm">
    ```bash
    pnpm add -g @caedonai/lord-commander
    ```
    
    Or for local installation:
    
    ```bash
    pnpm add -D @caedonai/lord-commander
    ```
  </Accordion>
  <Accordion icon="yarn" title="Install via yarn">
    ```bash
    yarn global add @caedonai/lord-commander
    ```
    
    Or for local installation:
    
    ```bash
    yarn add -D @caedonai/lord-commander
    ```
  </Accordion>
</AccordionGroup>

### Step 2: Verify Installation

<AccordionGroup>
  <Accordion icon="terminal" title="Check version">
    Verify that Lord Commander is installed correctly:
    
    ```bash
    lord-commander --version
    ```
    
    You should see the version number printed to the console.
  </Accordion>
  <Accordion icon="list" title="View available commands">
    Get a full list of available commands:
    
    ```bash
    lord-commander --help
    ```
    
    <Tip>Use `lord-commander <command> --help` for detailed help on any specific command.</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Initialize Your First Project

<AccordionGroup>
  <Accordion icon="folder-plus" title="Create a new project">
    Initialize a new project with Lord Commander:
    
    ```bash
    mkdir my-project
    cd my-project
    lord-commander init
    ```
    
    Follow the interactive prompts to configure your project.
  </Accordion>
  <Accordion icon="wand-magic-sparkles" title="Scaffold components">
    Generate components, pages, or other project assets:
    
    ```bash
    lord-commander scaffold component MyComponent
    lord-commander scaffold page HomePage
    ```
  </Accordion>
</AccordionGroup>
