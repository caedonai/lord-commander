# Scaffold Command

The `scaffold` command generates code using intelligent templates that follow your project's patterns and conventions. It's designed to speed up development by creating consistent, well-structured code.

## Basic Usage

```bash
# Generate a component
lord-commander scaffold component Button

# Generate a page
lord-commander scaffold page HomePage

# Generate an API endpoint
lord-commander scaffold api users
```

## Command Options

| Option | Description | Default |
|--------|-------------|---------|
| `--template` | Template to use | Auto-detected |
| `--path` | Output directory | Current directory |
| `--dry-run` | Preview without creating files | `false` |
| `--force` | Overwrite existing files | `false` |

## Examples

### React Component

```bash
lord-commander scaffold component Button --template typescript
```

Creates:
```
src/components/Button/
├── Button.tsx
├── Button.test.tsx
├── Button.module.css
└── index.ts
```

### API Route

```bash
lord-commander scaffold api users --crud
```

Creates:
```
src/api/users/
├── users.controller.ts
├── users.service.ts
├── users.model.ts
└── users.test.ts
```
