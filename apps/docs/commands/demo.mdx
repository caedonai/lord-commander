# Demo Command

The `demo` command provides interactive demonstrations of Lord Commander features. It's perfect for learning, onboarding new team members, or exploring capabilities.

## Basic Usage

```bash
# List available demos
lord-commander demo

# Run a specific demo
lord-commander demo init-project

# Interactive demo selection
lord-commander demo --interactive
```

## Demo Options

| Option          | Description                          | Default         |
| --------------- | ------------------------------------ | --------------- |
| `--interactive` | Interactive demo selection           | `false`         |
| `--fast`        | Skip delays and animations           | `false`         |
| `--output`      | Demo output directory                | `./demo-output` |
| `--clean`       | Clean up demo files after completion | `true`          |
