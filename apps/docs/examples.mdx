## Getting Started Examples

These examples show common Lord Commander workflows for different project types and scenarios.

## React Project Setup

### Initialize a React Project

```bash
# Create a new React project with TypeScript
mkdir my-react-app
cd my-react-app
lord-commander init --framework react --language typescript
```

**What this does:**

- Sets up React with TypeScript configuration
- Configures build tools (Vite/Webpack)
- Adds testing setup with Jest/Vitest
- Creates project structure with best practices

### Generate React Components

```bash
# Generate a basic component
lord-commander scaffold component Button

# Generate a component with props interface
lord-commander scaffold component Card --with-props

# Generate a page component
lord-commander scaffold page HomePage --route "/home"
```

## Vue Project Setup

### Initialize a Vue Project

```bash
# Create a Vue 3 project with Composition API
lord-commander init --framework vue --version 3 --composition-api
```

### Generate Vue Components

```bash
# Generate a single-file component
lord-commander scaffold component UserProfile

# Generate with composables
lord-commander scaffold component DataTable --with-composables
```

**Generated Vue component:**

```vue
<!-- UserProfile.vue -->
<template>
  <div class="user-profile">
    <h2>{{ user.name }}</h2>
    <p>{{ user.email }}</p>
  </div>

  padding: 1rem;

  border-radius: 0.5rem;
}
</style>
```

### CI/CD Integration

```yaml
# .github/workflows/bundle-analysis.yml
name: Bundle Analysis

on:
  pull_request:
    branches: [main]

jobs:
  analyze:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: "18"

      - name: Install dependencies
        run: npm ci

      - name: Run bundle analysis
        run: lord-commander analyze --ci --threshold 500kb

      - name: Generate report
        run: lord-commander analyze --report --format json > bundle-report.json

      - name: Upload report
        uses: actions/upload-artifact@v3
        with:
          name: bundle-report
          path: bundle-report.json
```

## Multi-Framework Workspace

### Monorepo Setup

```bash
# Initialize a monorepo with multiple frameworks
lord-commander init --workspace monorepo

# Add React app
lord-commander scaffold app frontend --framework react --location apps/

# Add Node.js API
lord-commander scaffold app api --framework node --location apps/

# Add shared library
lord-commander scaffold lib shared-utils --location libs/
```

### Cross-Framework Analysis

```bash
# Analyze entire workspace
lord-commander analyze --workspace

# Analyze specific apps
lord-commander analyze --apps frontend,api

# Generate workspace report
lord-commander analyze --workspace --report --output ./reports/
```

## Performance Optimization Examples

### Code Splitting Setup

```bash
# Analyze current bundle for splitting opportunities
lord-commander analyze --code-splitting

# Generate route-based splitting
lord-commander scaffold routes --lazy-loading

# Optimize third-party imports
lord-commander optimize --tree-shaking --chunk-splitting
```

### Progressive Enhancement

```bash
# Set up progressive loading
lord-commander scaffold pwa --service-worker

# Generate critical CSS
lord-commander optimize --critical-css

# Configure preloading strategies
lord-commander optimize --preload-strategy aggressive
```

## Testing Integration

### Generate Test Suites

```bash
# Generate unit tests for existing components
lord-commander scaffold tests --type unit --pattern "src/components/**/*.tsx"

# Generate integration tests
lord-commander scaffold tests --type integration --api-endpoints

# Generate E2E tests
lord-commander scaffold tests --type e2e --user-flows
```

### Test Configuration

```bash
# Set up testing environment
lord-commander init --testing jest,cypress

# Configure test coverage
lord-commander configure --coverage-threshold 80

# Generate test data factories
lord-commander scaffold factories --models User,Product,Order
```

## Deployment Examples

### Build Optimization

```bash
# Optimize for production
lord-commander build --optimize --analyze

# Generate deployment artifacts
lord-commander build --docker --k8s-manifests

# Create deployment package
lord-commander package --target vercel,netlify,aws
```

### Environment Configuration

```bash
# Set up multiple environments
lord-commander configure --env development,staging,production

# Generate environment-specific builds
lord-commander build --env production --optimize --minify
```

## Troubleshooting Examples

### Debug Bundle Issues

```bash
# Debug bundle composition
lord-commander analyze --debug --verbose

# Find circular dependencies
lord-commander analyze --circular-deps

# Identify unused dependencies
lord-commander analyze --unused-deps --prune
```

### Performance Debugging

```bash
# Profile build performance
lord-commander build --profile --benchmark

# Analyze build bottlenecks
lord-commander analyze --build-time --optimization-suggestions
```
